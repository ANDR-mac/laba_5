# laba_5
✔️ Кратко о работе программы

Программа позволяет работать со списком товаров, полностью используя базу данных.
Она умеет:
	•	добавлять обычные товары и товары со скидкой;
	•	редактировать и удалять товары;
	•	автоматически сохранять все изменения в таблицу SQLite (products);
	•	показывать товары в виде таблицы на экране;
	•	экспортировать данные из БД в JSON-файл;
	•	импортировать данные из JSON-файла обратно в БД;
	•	сортировать товары по конечной цене.

Таким образом программа обеспечивает хранение, редактирование и обмен данными товаров между БД и файлами.
